# HeroGameJS
Projeto de lógica para jogo como entrega no curso DIO - Lógica de Programação

Usando javascript puro.
O jogo cria classe para personagens, realiza interação com todos os itens do array herois simulando ataque com a arma de cada personagem, soma XP ao nível do personagem após cada interação.
Ao final é exibido o Resultado do ciclo.
